# tagiton.server

https://tagiron-client.vercel.app/ のサーバーサイド

# 起動

```
docker compose up
```

初回のみ

```
docker compose exec api yarn
```

# 使用技術

- Nestjs
- TypeORM
- MySQL
- Docker
